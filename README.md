# Server

커밋 컨벤션을 준수해주세요!

## 커밋 메시지 컨벤션

- **Feat**: 새로운 기능 추가
- **Refactor**: 코드 리팩토링
- **Chore**: 빌드 업무 수정, 패키지 매니저 수정, 잡다한 것
- **Fix**: 버그 수정
- **Docs**: 문서 수정
- **Style**: 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
- **Test**: 테스트 코드, 리팩토링 테스트 코드 추가
